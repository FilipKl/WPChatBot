# WPChatBot — AI-Powered Web Application

A web-based AI chatbot application developed as part of the *Web Programming (2025/2026)* course.

## Tech Stack
Java, Spring Boot, Spring AI, Ollama, MCP, SQLite, Thymeleaf, WebSockets, Spring Security

## Description
WPChatBot is an AI-powered chatbot that enables real-time conversations with persistent chat storage and secure user authentication.

The system integrates local LLM capabilities to provide context-aware responses and demonstrates full-stack web development concepts.

## Key Features
- AI-powered chatbot with local LLM integration (Ollama)  
- Real-time messaging using WebSockets  
- User authentication and security with Spring Security  
- Persistent chat storage using SQLite and Hibernate  
- REST API backend with Spring Boot  
- Thymeleaf-based web interface  

## Project Context
Developed as part of laboratory exercises for the *Web Programming (2025/2026)* course.
